# 应用方向

## 总原则

这条 idea 如果要落地，不能直接追“AGI 总方案”，而应该拆成小型、可测、可复现实验。

最强的落地方向不是：
- 做一个很像生物的描述系统

而是：
- 做一个在长期学习中可以增长、剪枝、迁移和重组能力的系统原型

## 方向一：可增长的持续学习系统

### 目标
研究一个模型在不断接收新任务时，能否：
- 保留旧能力
- 对新能力扩容
- 在资源受限下剪枝和重组

### 可做内容
- 固定架构 vs 可增长架构
- 手工扩容规则 vs 学习型扩容规则
- 带预算约束的容量分配

### 可观测指标
- 遗忘率
- 参数效率
- 新任务样本效率
- 推理和训练成本
- 模块复用率

## 方向二：学习规则自适应

### 目标
研究系统能否在不同阶段、不同模块上自动形成不同的更新策略。

### 可做内容
- 不同子模块使用不同更新频率
- 让元控制器决定何时强化、何时冻结、何时重学
- 让系统学会“什么时候应该探索新结构”

### 对应问题
这比“这块用 RL、那块用无监督学习”更深入，因为它在研究：
- 学习规则是否能被学习
- 结构决策是否能被学习

## 方向三：资源约束下的发展式智能

### 目标
把“神经元生命周期”翻译成资源优化问题。

### 可做内容
- 为模块设置计算预算
- 为记忆设置保留成本
- 用信息增益 / 任务贡献评估是否保留模块
- 实验“死亡”和“再分配”对整体性能的影响

### 这条线的价值
它能把你的生物直觉转成更清晰的计算问题：
- 什么值得活着
- 什么值得扩张
- 什么应该被遗忘

## 方向四：世界模型与主动交互

### 目标
检验“成长”是不是必须依赖主动探索，而不是只靠静态语料或静态数据集。

### 可做内容
- 在小型环境中训练世界模型
- 对比纯观察学习和主动交互学习
- 研究课程设计和能力阶段解锁

### 适用环境
- gridworld
- miniworld
- dm-control 的简单任务
- 自定义小型 sandbox

## 方向五：更长期的神经形态 / 仿生计算探索

### 目标
这条线不适合作为你马上开始的主线，但可以作为远期兴趣库。

### 可关注内容
- 脉冲神经网络
- 事件驱动计算
- 能耗约束
- 神经形态芯片生态

### 注意
这条线非常重，门槛也高，不建议一开始就把它当作主战场。

## 最推荐的近期落地方式

最值得先做的是：

`持续学习 + 可增长结构 + 资源预算`

原因：
1. 它最能承接你的第一条路线。
2. 它比“全脑模拟”现实得多。
3. 它能直接连到 RL、meta-learning、continual learning 的现有文献。
4. 它也更容易做出实验性结论，而不是停在宏大设想。
