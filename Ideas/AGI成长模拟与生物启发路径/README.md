# AGI成长模拟与生物启发路径

## 条目定位
这是一个围绕 AGI 的长期想法条目。核心不是简单“做得像生物”，而是追问：

1. 智能是否必须经历类似成长的阶段性发展？
2. 学习规则、网络结构、资源约束、环境交互是否应共同演化？
3. 现有“大模型 + 缩放”路线是否已经足够，还是缺了更深的结构性原则？

## 当前判断
Gemini 给出的很多方向判断其实是比较先进且准确的，尤其是：
- 把第一条路映射到 developmental learning / meta-learning / continual learning
- 把第二条路映射到 whole brain emulation / neuromorphic computing
- 强调资源约束、感觉运动闭环、动态拓扑的硬件困难
- 提出 Bitter Lesson 作为对“过度硬编码仿生机制”的警告

但还需要再往前走一步：
- 不能只停在“仿生 vs 缩放”的二分法
- 更核心的问题是：哪些生物原则是实现一般智能的必要计算约束，哪些只是自然界的实现细节
